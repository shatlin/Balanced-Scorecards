Competency Library


Overview

This module facilitates the career planning process for the individual, by helping them identify those jobs in the organization for which they have the competencies (skills and abilities) required to be successful in that position.

Benefits

  • Facilitates the career planning process
  • Allows the user to view all job profiles within the organization
  • Pre-loaded with fully validated and scaled competencies
  • Regularly update to provide you with constant and continuous improvements and best practices.

Features

Employees can view any job profile within the organization, including all skills, accountabilities and competencies attached to that profile, by proficiency level. Each proficiency level comes with a detailed list of the behavioural indicators that are required to function in that job.

Then, using the Assessment & Learning Manager, the individual can then conduct a detailed self-assessment against a specific job they have identified, and build a personalized learning plan to help them prepare for a new position in the organization.

If an organization’s has already developed their own set of competencies and proficiency levels, the Competency Dictionary can be loaded with these competencies instead.

The library features a comprehensive on-line help function which makes it easy-to-use for employees regardless of their technical background.
