Performance Planning Module


Overview

Empowering Employees through Effective Performance Management Systems

This module facilitates effective employee performance development through mutual access and input into an employee’s performance development (and/or learning) plan.

Benefits

  • Encourages open communication between the Manager and the employee
  • Facilitates the development of a mutually acceptable performance review

Features

Based on the employee’s self-assessment, a parallel supervisor assessment and the resulting gap analysis, the Manager (along with the employee) can identify specific skill areas for development.

Self-Assessment/Parallel Supervisor Assessment:

Managers dispatch a request for the completion of a self-assessment to employees. Once the individual has completed the assessment, the Manager is automatically notified via email. The Manager then completes a parallel assessment, and a Gap Analysis report is generated, comparing the two inputs.

Managers and employees then use this report for discussion, to develop a mutually acceptable performance review.

Performance Reviews:

Based on the gap analysis report, the Manager can set performance goals and objectives (with key indicators and outcomes), attach competencies to specific goals / objectives, and define behavioural examples for the key indicators. The employee’s success in displaying the desired behaviours can be rated as part of the review process.

The Manager then generates a draft performance review document and dispatches it to the employee. At this point, both the Manager and the employee have access for input and discussion, and ongoing adjustments to the plan can be logged.

Encouraging Ongoing Dialogue:

The Performance Plan encourages open communication between the Manager and the employee throughout the review period, as the plan may be amended or modified as necessary during the course of the year. And since goals and objectives are set based on open discussion, the performance plan (and evaluation) becomes more meaningful and motivational to the employee.
